Jellyfin Integration (Unofficial Repo)

An alternative setup using an unofficial webhook plugin from a community repository. Use this method if the official Jellyfin plugin repository is not accessible from your server.

Setup Instructions

  1. You will need to have TMDB and TVDB as metadata providers in Jellyfin.
  2. In Jellyfin, go to Dashboard > Plugins > Repositories and add the following repository URL:
    https://raw.githubusercontent.com/shemanaev/jellyfin-plugin-repo/master/manifest.json
  3. From the Plugin Catalog, install the Unofficial Webhooks plugin.
  4. Restart your server, then open the Unofficial Webhooks plugin and configure it:
    • Payload format: Default
    • Listen to events only for: your Jellyfin user
    • Events: Play and Stop

Kodi Integration

Connect Kodi to automatically track your watching progress.

Setup Instructions

  1. You will need TMDB and/or TVDB metadata sources enabled in Kodi so that external IDs are included in webhook payloads.
  2. Download and install the HTTP Scrobbler add-on.
  3. In Kodi, go to Settings > Services > Control and enable Allow remote control via HTTP.
  4. In the HTTP Scrobbler add-on settings, set the webhook URL to: Tip: open the Kodi web interface at http://<kodi-ip>:8080 to configure the add-on — easier than using the remote. {# djlint:off #}
    {% include "app/icons/copy.svg" with classes="w-5 h-5 text-gray-400 group-hover:text-indigo-400 transition-colors" %}
    {# djlint:on #}
  5. Leave the username and password fields blank.
  6. Enable the following events: Start, Stopped, and Finished.
  7. An item is marked watched when playback reaches 80% or Kodi reports a Finished event.

Controls whether Jellyseerr webhooks are allowed to add items to your Yamtrack list.

Examples: PENDING,PROCESSING,PARTIALLY_AVAILABLE,AVAILABLE. Leave blank for default behaviour (UNKNOWN is ignored).
